# AQChat_server

### 介绍
AQChat是基于Netty和ProtoBuf实现的匿名即时聊天室，支持文本、图片、文件、视频的发送和接收。
<br/>
项目中使用SpringBoot3整合Netty，并使用JDK17和ProtoBuf协议进行数据传输。

演示地址：https://demo.aqchat.run


### 项目架构说明
JDK17 <br/>
Netty 4.1.89.Final <br/>
SpringBoot 3.2.5 <br/>
Mybatis-Plus 3.5.3.2<br/>
protobuf 3 <br/>
MySQL 8.0.34 <br/>
Redis 6.0.8 <br/>
